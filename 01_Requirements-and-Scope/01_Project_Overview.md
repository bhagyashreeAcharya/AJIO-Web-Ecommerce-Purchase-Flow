# Project Overview

## Application Description

This project simulates end-to-end testing of an e-commerce web application inspired by AJIO.  
The focus is on validating core user flows from authentication to product purchase.

The system includes modules for:

- User Authentication (OTP-based login & registration)
- Product Browsing & Discovery
- Product Detail Page (PDP)
- Add to Bag & Cart Management
- Checkout & Order Flow
- Basic Account Management

---

## Business Objective

The primary objective of this project is to validate the core revenue-generating flow of an e-commerce platform:

User Login → Product Selection → Add to Cart → Checkout → Order Confirmation

Testing focuses on ensuring reliability, correctness, and traceability across this purchase journey.

---

## Testing Approach Summary

This project follows a structured Agile-style workflow:

- Epics → User Stories → Acceptance Criteria
- Test Scenarios derived from AC
- Detailed Test Cases
- Defect Logging
- RTM for traceability
- Test Execution Summary Report
